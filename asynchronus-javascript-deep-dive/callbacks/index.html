Callbacks

Callbacks are the traditionally way to implement async code in JS and avoid blocking code.
A callback is just a simple function that is execute when something happend.

How callbacks are possible?

In javascript exist objects we can define it, but also variables are object and maybe the most wear situation it´s that functions also are objects, this is very important in JS 'cause allow you to pass function to another functions (A function that receives another function trough param it´s called higher order function), and this allows you to execute a function inside another function.

Examples

A very basic example of callbacks are events, for example listen for a click in some button, we attach some function to the click event, and when click happend we execute it.
