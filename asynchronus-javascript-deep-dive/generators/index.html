Generators

Generators are functions that can return multiple values, one after another, on user demand, they works specialy good with iterations, allowing you to create data stream with ease.

How it look like?

we need to specify that a function is a generator using * in the function declaration.

In order to provide new values we need to use yield to provide values to consumers

    
      function *randomNumber(limit = 100) {
        while (true) {
          yield Math.floor(Math.random() * limit) + 1
        }
      }

How to consume?

In order to consume, we need to call the function to get a generator.

for consume values we need to call the function next provided by the generator

    
      console.log(randomNumber(100).next())

Important information

A generator cand send and also receive information from outside, this mean you can send data in every consume iteration and use it inside the generator this is so powerful way to work with generator. 

    
      *gen() {
        let dataFromOutside = yield "Send data please"
        console.log(dataFromOutside)
      }

      generator.next('Send value')
